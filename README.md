# 📢 EOS - Sistema de Avisos

Sistema de avisos para o **E.O.S - Um sistema operacional para a educação**.

## Funcionalidades
- API em Node.js + Express + SQLite
- Interface web leve e moderna
- Atualização automática de avisos
- Design escuro (ideal para XFCE)

## Como rodar

### 1. Iniciar a API

```bash
